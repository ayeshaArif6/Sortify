## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/ayeshaArif6/Sortify.git
    cd Sortify
    ```
2. Install dependencies:
    ```bash
    npm install
    ```
4. Start the development server:
    ```bash
    npm run dev
    ```
5. Open the application in your browser at `http://localhost:3137`.
